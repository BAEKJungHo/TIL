# Google Analytics Report

## 순서

1. 사이트를 구글 애널리틱스 콘솔에 등록
2. 구글 개발자 콘솔에서 새로운 프로젝트 생성
3. 프로젝트 관리자에서 서비스 계정 생성 -> 키파일 다운로드
4. 구글 애널리틱스 콘솔에 3번에서 생성된 계정을 사용자에 추가
5. API 호출하는 애플리케이션 코드 작성
