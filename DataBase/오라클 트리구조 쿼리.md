## 오라클 트리구조 쿼리

```sql
SELECT
    CATEGORY_SEQ,
    PARENT_CATEGORY_SEQ,
    TITLE,
    DATA,
    SORT,
    CONTENTS,
    LTRIM (SYS_CONNECT_BY_PATH (CATEGORY_SEQ,'>'),'>') AS depth_fullname
FROM
    MEC_C_CATEGORY
START WITH
    CATEGORY_SEQ = 'CAT073'
CONNECT BY
    PRIOR CATEGORY_SEQ = PARENT_CATEGORY_SEQ
```
