## 엑셀 다운로드

- jsp 
- apache poi
- jxls 

방식 등이 있다.

jxls 방식이 가장 느리고 jsp 방식이 가장 빠르다.

jsp 방식은 아래와 같다.

- list.jsp 에서 엑셀 다운로드 버튼 클릭 
- 컨트롤러에서 model 에 데이터 담아서 excel.jsp 로 리턴
- excel.jsp 에서 데이터만 테이블 형태로 뿌려주면 엑셀 다운로드 완료

```html

<%@ taglib prefix="fn" uri="http://java.sun.com/jsp/jstl/functions" %>

<table border="1">

            </tr>
        </c:if>
    </tbody>
</table>
```
